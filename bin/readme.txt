Clownacy's mod loader for the original Cave Story - Version 1.4.0.4

This comes with several examples mods. See their readmes for details.

To install this, just drag the 'mods' folder and the 'dsound.dll' file into Cave Story's folder.

You can find the source code for this at https://github.com/Clownacy/Cave-Story-Mod-Loader.

For more information, read this: https://www.cavestory.org/forums/threads/dll-mod-loader-ogg-music-mod.13879/
