This is a DLL mod loader for Freeware Cave Story. Also included in this repo are some example mods.

Dependencies:

Everything is designed to be compiled with MinGW-w64, particularly in an MSYS2 environment
https://www.msys2.org/

sdl_controller_input requires SDL2
https://libsdl.org/

ogg_music requires Cubeb and libvorbis
https://github.com/kinetiknz/cubeb
https://xiph.org/vorbis/
